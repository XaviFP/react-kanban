# Ongoing Kanban React-Node.js pet project

## How to run:

```sh
$ docker-compose up
```

## Running tests:

Server:

```sh
$ cd server/
$ npm test
```

Client:

```sh
$ cd client/
$ npm test
```

and follow the instructions.
